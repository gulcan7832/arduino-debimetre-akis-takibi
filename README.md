# arduino-debimetre-akis-takibi
Bu çalışma, YF-S201 Hall Effect sensörü kullanarak sıvı akış hızını (ml/dk) ve toplam tüketilen miktarı ölçen Arduino tabanlı bir sistemdir. Veriler, Arduino'nun Interrupt (kesme) özelliği ile yüksek hassasiyetle hesaplanarak seri port üzerinden takip edilmektedir.
